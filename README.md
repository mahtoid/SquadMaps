## What is SquadMaps?

**SquadMaps** is a web companion for [*Squad*](https://joinsquad.com) by Offworld Industries. Browse every map and layer, inspect factions, vehicles, tickets, and capture points - then enhance your match with tools built specifically for Squad players.

## Translations

SquadMaps is used by players worldwide. Translations are community-maintained - we need native speakers who know Squad terminology, not machine translations.

| Supported Languages | Locale | Notes |
|----------|--------|----------|
| 🇬🇧 English | `en` | Reference file |
| 🇩🇪 Deutsch | `de` | in active progress |
| 🇫🇷 Français | `fr` | in active progress |
| 🇷🇺 Русский | `ru` | in active progress |
| 🇨🇳 中文 | `zh-CN` | ✅ |
| 🇺🇦 Українська | `ua` | ✅ |

Progress is measured against `locale/en.json`. Read [CONTRIBUTING.md](CONTRIBUTING.md#translations), edit the locale file for your language, and open a pull request.

## About this Repository

This public repository is SquadMaps' community hub - translation files, issue tracking, and release notes. The live web app source is maintained separately and is not open-source today, though that may change. For now, **translations are the most impactful contribution you can make.**
